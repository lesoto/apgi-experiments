# Experimental Paradigms

This directory contains documentation for experimental paradigms in the APGI Framework.

## Available Paradigms

### 1. Consciousness Evaluation Tests
- **Primary Falsification Test** - Tests whether full ignition signatures can occur without consciousness
- **Secondary Falsification Test** - Tests specific consciousness markers

### 2. Cognitive Tasks
- **Interoceptive Gating** - Tests interoceptive precision and gating mechanisms
- **Cross-Species Validation** - Validates paradigms across different species

### 3. Neural Signature Analysis
- **P3b Component Analysis** - Analyzes P3b event-related potentials
- **Gamma Synchrony** - Measures gamma-band phase-locking
- **BOLD Activation** - fMRI activation pattern analysis

## Implementation Details

Each paradigm includes:
- Configuration parameters
- Data collection protocols
- Analysis methods
- Validation criteria
