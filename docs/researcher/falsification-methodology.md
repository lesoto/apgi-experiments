# APGI Framework Falsification Test: Defining the Boundaries

The APGI Framework is decisively falsified by replicable demonstrations of specific dissociations between neural signatures and conscious access. Below are the exact criteria for falsification, organized by priority and scope.

## Primary Falsification Criterion

### Full Ignition Signatures Without Consciousness

The framework is falsified if the following occurs in multiple independent labs using converging methods:

#### Required Signatures (ALL must be present)

| Signature | Threshold |
| --- | --- |
| P3b ERP | Amplitude >5 μV at Pz electrode (relative to prestimulus baseline), latency 250–500 ms |
| Gamma-band synchrony | Phase-locking value (PLV) >0.3 between frontoparietal and temporal regions, sustained >200 ms (30–80 Hz) |
| BOLD activation (fMRI) | Z > 3.1, cluster-corrected p < 0.05, encompassing: |

- • Bilateral dorsolateral prefrontal cortex
- • Intraparietal sulcus
- • Anterior insula or anterior cingulate cortex

| Signature | Threshold |
| --- | --- |
| Perturbational Complexity Index | PCI > 0.4 (measured via TMS-EEG; Casali et al., 2013) |

#### Required Absence (BOTH must be absent)

| Absence | Verification Method |
| --- | --- |
| Conscious reportability | • Subjective report ("Did you see/hear X?")—consistently negative |
| | • Objective forced-choice (above-chance performance)—at chance level |
| | • Post-perceptual measures (confidence ratings, wagering)—no evidence of access |
| AI/ACC engagement | • No significant BOLD response in anterior insula/ACC (same threshold as above) |
| | • Gamma-band coherence between AI/ACC and frontoparietal cortex: PLV ≤ 0.25 |
| | • No effective connectivity (DCM/Granger causality) showing AI/ACC drives frontoparietal activity (0–500 ms) |

#### Control Requirements

- Participants must report other stimuli (intact motor/verbal systems).
- Tasks must be well-trained and understood.
- Stimuli must be supraliminal in control conditions.
- Multiple converging measures of consciousness (not relying solely on verbal report).
- Participants must demonstrate intact metacognitive sensitivity (e.g., confidence-accuracy correspondence) to rule out cases of unconscious processing with impaired reportability due to executive dysfunction rather than absent ignition.

#### Interpretation of Partial Signatures

- Not falsification: Partial signatures (e.g., late frontal negativity without P3b, local gamma without long-range synchrony).
- Interpretation: Subthreshold or failed ignitions, consistent with the framework.
- Only the complete pattern above constitutes falsification.

## Secondary Falsification Criteria

### Criterion 2: Consciousness Without Ignition Signatures

Hypothesis:

If verified conscious reports occur without any ignition signatures, the framework requires major revision.

Operationalization:

| Requirement | Threshold |
| --- | --- |
| Conscious report | Verified via forced-choice, confidence ratings, or wagering |
| Absence of signatures | • P3b < 2 μV |
| | • Gamma PLV < 0.15 |
| | • PCI < 0.3 |
| | • No frontoparietal BOLD elevation |
| Occurrence | >20% of trials across multiple participants |

Possible Interpretations:

- Alternative ignition pathways (e.g., corticocortical loops bypassing thalamus).
- Measurement sensitivity issues (signatures present but below detection).
- Conscious access without full workspace broadcast (e.g., fringe consciousness).

True Falsification:

Consciousness occurs without ignition and without any alternative mechanism for global broadcast (e.g., preserved consciousness with complete frontoparietal network lesions).

### Criterion 3: Threshold Insensitivity to Neuromodulation

Hypothesis:

If the ignition threshold (θₜ) shows no modulation by neuromodulators despite pharmacological manipulation, the neuromodulatory implementation is falsified.

Operationalization:

| Pharmacological Challenge | Expected Effect | Falsification Condition |
| --- | --- | --- |
| Propranolol (NE blockade) | θₜ modulation | θₜ unchanged |
| L-DOPA (DA elevation) | θₜ modulation | θₜ unchanged |
| SSRIs (5-HT elevation) | θₜ modulation | θₜ unchanged |
| Physostigmine (ACh elevation) | θₜ modulation | θₜ unchanged |

Threshold Estimation:

- Detection threshold tasks.
- P3b amplitude (inverse relationship).
- Computational modeling of behavioral data.

Control:

Drug effects verified via pupil response, physiology, or cognitive performance.

Implication:

Suggests θₜ is not implemented through neuromodulation, requiring an alternative mechanism (e.g., structural network properties alone).

### Criterion 4: Soma-Bias Absence

Hypothesis:

If interoceptive and exteroceptive prediction errors show no baseline difference in ignition probability when matched for precision (β ≈ 1.0), the soma-bias claim is falsified.

Operationalization:

- Design tasks with matched Πₑ and Πᵢ (verified independently).
- Compare:
  - Detection thresholds (interoceptive vs. exteroceptive).
  - P3b amplitude to matched-precision stimuli.
  - Reaction times.
  - Attentional blink magnitude.
- Require a large sample (n > 100) to detect β if close to 1.

Expected vs. Falsification:

| Condition | Expected β | Falsification β |
| --- | --- | --- |
| Neutral contexts | 1.2–1.5 | 0.95–1.05 |

Requires removing soma-bias from the core framework, simplifying to:

Sₜ = Πₑ·|εₑ| + Πᵢ·|εᵢ| (no weighting difference).

## Edge Cases and Boundary Conditions

| Case | APGI Interpretation | Falsification? |
| --- | --- | --- |
| Anesthesia Awareness | Minimal/fragmented ignition or alternative pathway (corticocortical). | No, unless full signatures without reportability. |
| Blindsight | Local sensory processing without ignition (Sₜ < θₜ). | No. |
| Dreams | Ignition with altered threshold dynamics, reduced frontal engagement, impaired memory encoding. | Partially within scope. |
| Locked-In Syndrome | Intact ignition; motor execution pathway damaged. | No; supports framework. |

## Framework Falsifiability Summary

The APGI Framework is structurally falsifiable through:

- Direct dissociation: Full ignition without consciousness (primary criterion)
- Reverse dissociation: Consciousness without ignition (secondary criterion)
- Mechanism failure: Neuromodulatory insensitivity or absent soma-bias (tertiary criteria)

Not falsified by:

- Partial signatures (subthreshold dynamics)
- Edge cases with alternative explanations (dreams, blindsight, anesthesia awareness)
- Failures of specific predictions if the core mechanism remains supported

Framework falsifiability is ensured by the presence of multiple, independent criteria that can be used to test the framework's core claims.
