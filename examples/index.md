# APGI Framework Examples

This directory contains comprehensive examples demonstrating how to use the APGI Framework.

## Available Examples

### Core Testing Examples

- **01_run_primary_falsification_test.py** - Primary falsification test with various configurations
- **02_batch_processing_configurations.py** - Batch processing with multiple parameter sets
- **03_custom_analysis_saved_results.py** - Custom analysis of saved test results
- **04_extending_falsification_criteria.py** - Extending falsification test criteria

### Framework Examples

See the `framework_examples/` subdirectory for detailed framework usage examples.

### Parameter Validation

- **parameter_validation_example.py** - Demonstrates parameter validation and error handling

### Validation and Error Handling

- **validation_and_error_handling_example.py** - Comprehensive validation and error handling examples

## Running Examples

Each example can be run independently:

```bash
# Run primary falsification test
python examples/01_run_primary_falsification_test.py

# Run batch processing
python examples/02_batch_processing_configurations.py

# Run custom analysis
python examples/03_custom_analysis_saved_results.py
```
